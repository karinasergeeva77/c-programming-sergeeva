## Задание 1.2: Сумма двух чисел
### Постановка задачи
Напишите программу, которая запрашивает два целых числа и выводит их сумму.
### Математическая модель
Сумма двух чисел вычисляется по формуле:
S=a+b
где $a$ и $b$ — введённые пользователем числа.
### Список идентификаторов
| Имя | Тип | Описание |
|-----|-----|----------|
| `a` | `int` | Первое число (ввод пользователя) |
| `b` | `int` | Второе число (ввод пользователя) |
| `sum`| `int`| Результат сложения |
### Код программы
```c
#include <stdio.h>

int main(void) {
    int a, b, sum;
    
    printf("Enter first number: ");
    scanf("%d", &a);
    
    printf("Enter second number: ");
    scanf("%d", &b);
    
    sum = a + b;
    
    printf("Sum: %d + %d = %d\n", a, b, sum);
    
    return 0;
}
```

## Задание 1.3: Произведение двух чисел
### Постановка задачи
Напишите программу, которая запрашивает два целых числа и выводит их произведение.
### Математическая модель
Произведение двух чисел вычисляется по формуле:
P=a×b
где $a$ и $b$ — введённые пользователем числа.

| Имя | Тип | Описание |
|-----|-----|----------|

| `b` | `int` | Второе число (ввод пользователя) |
| `product`| `int`| Результат умножения |

```c
#include <stdio.h>

int main(void) {

    scanf("%d", &a);
    
    printf("Enter second number: ");
    scanf("%d", &b);
    
    product = a * b;
    
    printf("Product: %d * %d = %d\n", a, b, product);
    
    return 0;
}
```

## Задание 1.4: Площадь круга
### Постановка задачи
Напишите программу, которая запрашивает радиус круга и вычисляет его площадь.
### Математическая модель
Площадь круга вычисляется по формуле:
S=πr^2
где $r$ — радиус круга, $\pi \approx 3.14159$.
### Список идентификаторов
| Имя | Тип | Описание |
|-----|-----|----------|
| `radius` | `double` | Радиус круга (ввод пользователя) |
| `area` | `double` | Вычисленная площадь круга |
| `PI`| `const double`| Число π (3.14159) |
### Код программы
```c
#include <stdio.h>

int main(void) {
    double radius, area;
    const double PI = 3.14159;
    
    printf("Enter radius: ");
    scanf("%lf", &radius);
    
    area = PI * radius * radius;
    
    printf("Area: %.2lf\n", area);
    
    return 0;
}
```

## Задание 1.5: Конвертер температуры
### Постановка задачи
Напишите программу, которая переводит температуру из градусов Цельсия в градусы Фаренгейта.
### Математическая модель
Перевод температуры выполняется по формуле:
F=C×9/5+32
где $C$ — температура в градусах Цельсия, $F$ — температура в градусах Фаренгейта.
### Список идентификаторов
| Имя | Тип | Описание |
|-----|-----|----------|
| `celsius` | `double` | Температура в градусах Цельсия (ввод пользователя) |
| `fahrenheit` | `double` | Температура в градусах Фаренгейта (результат) |
### Код программы
```c
#include <stdio.h>

int main(void) {
    double celsius, fahrenheit;
    
    printf("Enter temperature in Celsius: ");
    scanf("%lf", &celsius);
    
    fahrenheit = (celsius * 9 / 5) + 32;
    
    printf("Temperature in Fahrenheit: %.2lf\n", fahrenheit);
    
    return 0;
}
```
